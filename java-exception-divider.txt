The class Divider.java is given below.
class Divider {

    private int A, B;

    Divider(int A, int B) {
        this.A = A;
        this.B = B;
    }

    int divide() {
        return A / B;
    }
}
Please fill in the missing lines of code for the main method class Hello.java so that the program prints the output as mentioned below.
